# Mythological Trivial

**Mythological Trivial** is a quiz game designed to test your knowledge of Greek and Norse mythology. Through a simple and fun interface, players must answer a series of questions related to gods, monsters, and mythical legends. Each correct answer adds to the score, and a final message shows the player's total points.


This project aims to both entertain and educate, encouraging learning about ancient cultures in an interactive way. It can also serve as a base for more complex projects or educational tools.

---

## Features

### Navigation
- The game runs as a single-page application with a clear and minimalistic layout.
- Upon loading, users see a message prompting them to start the game.
- Answer buttons update dynamically with each new question.
- At the end of the game, the final score is shown and users are invited to play again.

## Code Structure

### HTML
- **Header**: Displays the game title.
- **Container**: Holds the score, current question, answer buttons, and start button.
- **Footer**: Contains author information.

### CSS
- Clean design with ancient-inspired colors.
- Responsive layout and readable typography.
- Interactive buttons with hover effects.

### JavaScript
- Handles game logic: questions, answers, score tracking, and feedback.
- Questions stored in an array of objects.
- Manages click events for both starting the game and selecting answers.
- Displays alerts for correct/incorrect answers and final score.

---

## Key Functionalities

- Semantic HTML tags for better accessibility and structure.
- Separation of concerns: HTML for structure, CSS for design, JS for logic.
- Fully client-side interaction, no backend required.
- Modular and easy to expand (e.g., more questions or mythology categories).
